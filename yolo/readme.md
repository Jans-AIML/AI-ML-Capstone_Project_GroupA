# YOLOv11s Training on COCO 2017 (From Scratch)

## Step-by-Step Setup

### 1. Install Dependencies

```bash
pip install ultralytics tqdm
```

### 2. Download COCO 2017 Dataset

Download from https://cocodataset.org/#download:

### 3. Convert Annotations to YOLO Format

Edit `COCO_DIR` in `convert_coco_to_yolo.py`, then:

```bash
python convert_coco_to_yolo.py
```

### 4. Update Paths in `coco.yaml`

Edit the `path` field to point to your dataset root:

```yaml
path: /home/user/datasets/coco # <-- update this
```

### 5. Verify Directory Structure

```
/home/user/datasets/coco/
├── images/
│   ├── train2017/          # 118,287 images
│   └── val2017/            # 5,000 images
├── labels/
│   ├── train2017/          # 118,287 .txt files (generated by converter)
│   └── val2017/            # 5,000 .txt files
├── annotations/
│   ├── instances_train2017.json
│   └── instances_val2017.json
└── coco.yaml
```

### 6. Train

```bash
python train_yolov11m_coco.py
```

---

## Files

| File                      | Description                                        |
| ------------------------- | -------------------------------------------------- |
| `coco.yaml`               | Dataset config with paths and 80 class names       |
| `convert_coco_to_yolo.py` | Converts COCO JSON annotations to YOLO .txt format |
| `train_yolov11m_coco.py`  | Main training script with GPU accelerated devices  |
